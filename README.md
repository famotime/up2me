## 使用教程

### 基础使用

1. 以管理员身份运行程序
2. 从下拉列表选择目标游戏进程
3. 点击"附加"连接到进程
4. 在游戏中找到要修改的数值
5. 选择适当的数值类型（整数/浮点数）
6. 输入当前值进行首次扫描
7. 在游戏中改变目标值
8. 使用"下一次扫描"缩小范围
9. 找到目标地址后可以锁定数值

### 高级搜索技巧

1. 数值类型选择：
   - 整数(4字节)：适用于等级、金钱等整数值
   - 浮点数：适用于坐标、速度等小数值
   - 双精度：适用于需要高精度的数值

2. 搜索模式：
   - 精确匹配：完全匹配输入值
   - 大于/小于：搜索范围值
   - 已改变：数值发生变化
   - 未改变：数值保持不变

3. 定位技巧：
   - 使用多次扫描逐步缩小范围
   - 观察数值变化规律
   - 注意相关联的数值
