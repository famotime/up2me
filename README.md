## 使用教程

### 基础使用

![image-20250222232307974](./image/image-20250222232307974.png)

1. 从下拉列表选择目标游戏进程；
2. 点击"附加"连接到进程；
3. 在游戏中找到要修改的数值（选择适当的数值类型），填入搜索栏；
4. 进行首次扫描；
5. 在游戏中改变目标值，并使用"下一次扫描"缩小范围；
6. 找到目标地址后“添加”到结果栏修改或锁定数值；

![image-20250222232741668](./image/image-20250222232741668.png)

### 搜索技巧

1. 数值类型选择：
   - 整数(4字节)：适用于等级、金钱等整数值
   - 浮点数：适用于坐标、速度等小数值
   - 双精度：适用于需要高精度的数值

2. 搜索模式：
   - 精确匹配：完全匹配输入值
   - 大于/小于：搜索范围值
   - 已改变：数值发生变化
   - 未改变：数值保持不变

3. 定位技巧：
   - 使用多次扫描逐步缩小范围
   - 观察数值变化规律
   - 注意相关联的数值
